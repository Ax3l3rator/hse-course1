## Setup

> [App installer](https://disk.yandex.ru/d/UgMk-l47FTomgQ)

---

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
